# Hash Table
要了解hash table，首先要熟悉dictionary的概念     
>## dictionary
>>Dictionary是以「鍵值-資料對」(Key-Value pair)來描述資料的抽象資料形態(Abstract Data Type)。      
>>例子：       
>>-電話簿裡的Dictionary即是將「姓名」視為Key，「電話號碼」視為Value。       
>>-學校學籍系統的Dictionary將「學號」視為Key，「學生資料」(如姓名、修課記錄)視為Value。
